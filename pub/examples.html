Basic Usage

This is the basic word cloud generated from the following coronavirus analysis text. By clicking on a word in the cloud, it will be highlighted in the text. This can be useful for highlighting key words such as "coronavirus" or "vaccine" to highlight parts you want to see.

Coronavirus Disease Vaccine Analysis

An analysis of the current coronavirus disease in Canada has been conducted. The vaccines that are currently approved for use in Canada are Pfizer-BioNTech, Moderna, AstraZeneca, COVISHIELD, and Janssen. Janssen is the only vaccine that is given in one dose. The coronavirus vaccination began in Canada in the week of December 13, 2020. Over 9 million first doses of the vaccine have been administered which is around 20% of the total population, while just under 1 million have received their second dose of the vaccine. We know that one dose of the vaccine has an 80% effectiveness against the coronavirus disease. For more coronavirus vaccine updates, subscribe to our newsletter.
const cloudy1 = new Cloudy();
cloudy1.title = "Word Cloud for Coronavirus Vaccine Analysis"
cloudy1.id = "#example1";
cloudy1.selectors.push("#text1");
cloudy1.generateWordCloud();

Dynamic Word Cloud Generation

One of the most common usages of word clouds is to be able to be generated dynamically based on user input. Dynamic word clouds can be prepopulated with data. Try adding a thought about your future into the word cloud!
const cloudy2 = new Cloudy();
cloudy2.title = "Thoughts of Your Future";
cloudy2.id = "#example2";
cloudy2.largest = 2;
const thoughts = ["scared", "work", "work", "family", "family", "scared", "cool", "long", "school", "summer",
"work", "family", "excited", "cool", "family", "work", "long", "scary", "california", "life"]
cloudy2.generateDynamicWordCloud(thoughts);
<div id="example2"></div>

Text Analysis

This is a text analysis generated from a sample student essay. From this analysis, we can see patterns such as the student using words such as React and learned extremely frequently. After analyzing the results, the student can make edits to their paper accordingly. Because we are generating the analysis within a container, we can style the container separately, for example, let's push our text analysis to the left side.

CSC309 Course Review

This winter semester, I have decided to take the course CSC309, Programming on the Web. From this course, I learned that web development is extremely difficult! Therefore, I will spend more time this summer learning more about web development and libraries like React. I also learned about React libraries such as React Bootstrap and more things about React. One thing that I really liked about React was how easy it was to use. I think my future projects will involve more React over other libraries and frameworks. I also learned about other backend frameworks such as Express.js and how to create a server. Next year I want to learn about Angular and Vue!
